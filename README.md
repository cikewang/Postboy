# Postboy
Http请求模拟工具、支持批量上传、一键批量测试

### 本地测试
~~~
# python manage.py runserver
~~~

访问URL：http://127.0.0.1:9999/page/
	

### 使用版本：
* Python 3.6
* Django 2.0.1
* requests 2.19
* xlrd 1.1
